Promotional Value Modeler

An analytical engine for modeling and quantifying the true cash value of complex casino promotions.

The Analytical Challenge: Quantifying Value

I was initially drawn to casino bonus promotions because of how generously they seemed to be giving out free money. I quickly learned that the actual value of each promotion is not immediately apparent from its brief description and that $1 of casino bonus money is not ever really worth $1 of cash.

Finding the actual value of a casino promotion is difficult to do and involves perfectly interpreting terms and conditions, understanding wagering requirements, and familiarizing yourself with each platform's own unique rules. The challenge was to create a tool that could model the entire system to create a plan that delivers the promotion's true cash value.

The Solution: A Prescriptive Modeling Engine

I first built a spreadsheet for deposit bonuses but later adapted this to a React app so it could handle other more complex bonus types. The final product is a complex calculator that takes the parameters of one or two different casino promotions and outputs a strategy for the optimal way to complete the offer.

The result is something magic and speaks for itself. It does not provide a guess nor does it recommend a 'betting strategy' where results are still affected by random chance and luck. Instead, it offers a complete plan of how much to deposit, how much to wager and on exactly what outcomes, and an exact, quantified outcome one can expect from executing the strategy correctly.

Project Details

This tool evolved from a simple Excel spreadsheet for deposit bonuses into a standalone React application. I iteratively rebuilt it to handle more complex optimization logic, manage multi-promo scenarios, solve for complex gift card scaling, and implement an "Expert Mode" analysis.

Stack: React, JavaScript (Hooks), Babel, Tailwind CSS

Core Concepts: Prescriptive Modeling, Constrained Optimization, Equilibrium Analysis, System Simulation

Status: Personal R&D (Unpublished)

Excel Spreadsheet

The first version was a simple spreadsheet that could only model a single deposit bonus at a time.

First Web Version

Migrated the logic to a React/JS app. This version incorporated a key analytical insight that significantly improved the model's potential.

React Migration

Refined the React app, implementing iterative solving techniques that would have been too clunky for Excel, solidifying the single-bonus logic.

Lossback & Dual-Promo

Expanded the engine to model 'Lossback' bonuses and, crucially, to solve for an optimal plan leveraging *two* promotions at the same time.

Gift Card Integration

Added logic to handle fixed-value inputs like casino gift cards, which introduced new and complex scaling challenges for the optimization engine.

Expert Mode

Instead of one "optimal" solution, I developed an "Expert Mode" that models and presents six different viable strategies (e.g., breakeven, max value) for a single scenario.

Current Version

The current version includes nuanced logic for handling Baccarat's "Tie" outcome, representing the most recent stable build.

Future Development

This project is still an active intellectual exercise. I'm currently modeling the impact of debit/credit card cashback, and I plan to explore adapting this same prescriptive methodology to other complex, multi-variable games like Craps.

Project Philosophy & Motivation

Practically speaking, this was just a fun exercise motivated by my desire to better understand casino bonuses. I was far more interested in the challenge of building the model than I was in ever using it in practice.

I get my enjoyment from *solving the puzzle* and building the analytical engine. I'm an analyst at my core, not a player.
